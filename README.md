# Toe-tac-tic-games
Next level of tic tac toe game, with movement for pieces

Cara Mendownload Game
1. Download file git Hub
2. Ekstrak File Zip
3. Mainkan game pada html 

Cara memainkan Game
1. Letakkan piece (X atau O) pada board seperti tic tac toe biasa sampai tiga piece untuk masing-masing X dan O
2. Setelah tiga piece, gerakkan piece dengan memilih piece yang ingin di gerakkan dan klik kotak yang kosong. Piece berhasil dipindahkan
3. Pindahkan piece sampai membentuk garis (baik horizontal, vertikal, atau diagonal)
